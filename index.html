Basic Terms

Whats is table or relation in database?

  • A table is a collection or related data held in a structure format whitin a base.

insert

insert data into a table called "users"

insert into users( name, idade )
values
( "Nome 1", "20" ),
( "Nome 2", "25" ),
( "Nome 3", "30" );

select

fetches the data that is already recorded in the table

select tye whole table

select * from users;

select a specific column in a table

select name from users;
-- or
select name, idade from users;

select * from users where idade = 13;
-- or
select * from users where idade > 13 and genre = f;

update

is used to change data that has already been inserted in the table

update users set nome = "Pessoa 2" where id = 2;

alter

used to alter created tables, you can add or delete or rename a column,
